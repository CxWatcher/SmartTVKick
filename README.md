# SmartKickTV
Smart Kick TV is a simple app to watch the Kick streaming service for FireTV &amp; Android TV devices

### How to use
1. Install APK (Instructions Below)
2. Search Your Streamer
3. Relax and Enjoy!

### Controls
- Press OK To Search / Play / Pause
- Press Next or Back to Close Keyboard
- Press Back to exit out of a Live Stream
- Press Menu for Quality Controls
- Use DPAD for up and down Streamer Selection

### How to install
#### Easy

1. Install [Downloader by AFTVnews](https://www.aftvnews.com/downloader/) from [Google Play](https://play.google.com/store/apps/details?id=com.esaba.downloader) or [Amazon app store](https://www.amazon.co.uk/dp/B01N0BP507)
2. Enter this code inside Downloader app: **9459513** and download the apk
3. Install the app, launch it, search your favorite streamer and enjoy!

#### Medium (Device)
1. Download the [Latest APK](https://github.com/CxWatcher/SmartTVKick/releases/download/0.0.1/SmartKickTV-v0.0.1.apk) onto your device
2. using a file browser select and install the apk
3. launch it, search your favorite streamer and enjoy!

#### Hard (PC)
1. Download the [Latest APK](https://github.com/CxWatcher/SmartTVKick/releases/download/0.0.1/SmartKickTV-v0.0.1.apk) onto your device or pc
2. Install and setup ADB
3. On Your FireTV / Android Device go to Developer Settings and Enable **ADB Debugging**
4. Find your Devices IP Address
5. Connect to your device ```adb connect YOURIPHERE```
6. If promted on your TV accept the connection
7. Install the APK ```adb install SmartKickTV-V0.0.1.apk```
8. launch it, search your favorite streamer and enjoy!
